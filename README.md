# Movie Recommendation System

## Project Description
A Machine Learning based movie recommendation system that suggests similar movies based on user-selected movies.

## Algorithm Used
Content Based Filtering

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Working Process

1. Movie dataset is loaded
2. Important movie features are combined
3. Text data is converted into numerical vectors
4. Cosine similarity is calculated
5. Similar movies are recommended

## How to Run

streamlit run app.py
